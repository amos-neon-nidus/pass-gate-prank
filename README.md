# Pass Gate Prank

一个无需后端的中文图片验证整蛊页面。用户可以选择九宫格图片、刷新题目并提交验证；无论选择什么，页面都会给出一本正经的“您是人机”结果。

## 特性

- 三组离线图片题库，每组 9 张，重试时切换到下一组。
- 九宫格选择、普通刷新、失败弹窗和重试交互。
- 提交后依次展示访问检查、挑战检查与完成验证状态。
- 所有图片和脚本均随项目提供，不依赖 CDN、接口或账号登录。
- 支持直接双击 `index.html` 离线打开。

## 本地打开

直接在浏览器中打开 `index.html` 即可。也可以在项目目录运行任意静态文件服务器，例如：

```bash
python -m http.server 8000
```

然后访问 `http://localhost:8000`。

本项目仅用于娱乐和前端交互演示，不会收集、传输或验证用户的真实个人信息。
